# Erreur de lecture Code 5001 sur l'application iOS

**Severity:** LOW

## Description

Je rencontre une erreur de lecture persistante sur l'application iOS (version 2.4.1) sur mon iPhone 13 Pro fonctionnant sous iOS 17.2. Quand j'essaie de lire du contenu, j'obtiens le code d'erreur 5001 et le message 'La lecture a échoué. Veuillez réessayer.' Cela se produit avec le contenu téléchargé et diffusé. J'ai essayé : désinstaller et réinstaller l'application, redémarrer mon téléphone, effacer les données de l'application, mettre à jour vers la dernière version iOS, et basculer entre WiFi et données cellulaires. Rien ne fonctionne. La version web fonctionne bien sur le même appareil en utilisant Safari. Cela a commencé après la dernière mise à jour de l'application (2.4.1) que j'ai installée il y a 5 jours.
